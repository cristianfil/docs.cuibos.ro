# Vehicule

Modulul de vehicule te ajută să ții evidența completă a mașinilor familiei: date tehnice, cheltuieli, kilometraj, mentenanță și alerte de conformitate.

![Pagina de vehicule cu cardul Dacia Duster](/screenshots/vehicles.png){width="800"}

## Adăugarea unui vehicul

Din sidebar → **Vehicule** → butonul **+** (adaugă vehicul).

### Câmpuri disponibile

| Câmp | Obligatoriu | Descriere |
|---|---|---|
| **Marcă** | ✓ | Ex: Dacia, Volkswagen, BMW |
| **Model** | ✓ | Ex: Duster, Golf, Seria 3 |
| **An** | ✓ | Anul fabricației |
| **Nr. înmatriculare** | ✓ | Ex: B-123-ABC |
| VIN | ✗ | Vehicle Identification Number (seria de șasiu) |
| **Combustibil** | ✓ | Benzină, diesel, electric, hibrid, GPL |
| Capacitate motor | ✗ | Capacitatea în cm³ (ex: 1998) |
| Culoare | ✗ | Culoarea vehiculului |

## Profilul vehiculului

Fiecare vehicul are un **profil complet** care afișează:

- **Date tehnice** — nr. înmatriculare, VIN, an, capacitate motor
- **Odometru** — kilometrajul curent (editabil)
- **Date de conformitate** — cu butoane de editare rapidă:
  - **RCA** — data expirării + nr. poliță
  - **Rovinieta** — data expirării
  - **ITP** — data expirării

::alert{type="warning"}
**Important:** Completează datele de expirare pentru ITP, RCA și Rovinieta — CuibOS trimite alerte automate cu 30, 14 și 7 zile înainte de expirare.
::

## Cheltuieli vehicul

Înregistrează orice cheltuială legată de vehicul. Tipuri uzuale:

| Tip | Exemple |
|---|---|
| **Combustibil** | Alimentări cu cantitate (litri), preț/litru, km |
| **Service** | Reparații, revizii, diagnoze |
| **Asigurare** | RCA, CASCO |
| **Taxe** | ITP, impozit auto, rovinieta |
| **Altele** | Spălătorie, parcare, amenzi, accesorii |

Fiecare cheltuială se salvează cu **data**, **suma** și opțional **note** descriptive.

## Kilometraj (Odometru)

Înregistrează periodic kilometrajul vehiculului apăsând pe iconița de editare de lângă odometru. CuibOS:
- Păstrează **istoricul** citirilor de kilometraj
- Calculează **consumul mediu** pe baza alimentărilor și km parcurși

## Plan de mentenanță

Secțiunea dedicată mentenanței planificate permite definirea operațiunilor recurente:

### Adăugarea unei operațiuni

Completează:
- **Numele** — ex: „Schimb ulei", „Schimb anvelope", „Revizie periodică"
- **Tip recurență** — după dată sau după km
- **Data scadenței** — când trebuie efectuată
- **Detalii** — informații suplimentare

### Jurnalul de mentenanță

Fiecare operațiune de mentenanță are un **jurnal** (log) cu istoricul execuțiilor: data efectuării, km la momentul respectiv, costul și notele. Poți comuta între vizualizarea **Plan** și **Jurnal**.

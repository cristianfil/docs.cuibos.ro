# Proprietăți

Modulul de proprietăți îți permite să administrezi toate imobilele familiei: apartamente, case, garaje, birouri, terenuri etc.

## Adăugarea unei proprietăți

::steps

### Accesează secțiunea Proprietăți

Din meniul lateral, alege **Proprietăți** → **Adaugă proprietate**.

### Completează detaliile

Câmpuri disponibile:

| Câmp | Obligatoriu | Descriere |
|---|---|---|
| Nume | ✓ | Numele proprietății (ex: „Apartament Centru") |
| Tip | ✓ | house, apartment, studio, garage, office, shop, workshop, land, other |
| Adresă | ✗ | Adresa completă |
| Oraș | ✗ | Orașul |
| Județ | ✗ | Județul |
| Cod poștal | ✗ | Codul poștal |
| Suprafață | ✗ | Suprafața utilă (mp) |
| Camere | ✗ | Numărul de camere |
| Băi | ✗ | Numărul de băi |
| Descriere | ✗ | Note suplimentare |

### Salvează

Proprietatea va apărea pe card-ul din dashboard cu statusul **activ**.

::

## Statusuri proprietate

- **Activ** — proprietate folosită în mod curent
- **Închiriat** — proprietatea este dată în chirie
- **Inactiv** — proprietate temporar neutilizată

## Blocarea proprietății

Proprietarul contului (owner) poate **bloca** o proprietate pentru a preveni modificările accidentale din partea celorlalți membri ai echipei.

::alert{type="info"}
O proprietate blocată poate fi vizualizată, dar nu poate fi editată sau ștearsă de nimeni cu excepția owner-ului.
::

## Ștergerea proprietății

Ștergerea este disponibilă doar pentru rolurile **owner** și **admin**. Proprietatea nu dispare definitiv — este marcată ca ștearsă (soft delete) și poate fi recuperată de echipa de suport.

# Proprietăți

Modulul de proprietăți îți permite să administrezi toate imobilele familiei. Fiecare proprietate are propriul card de prezentare, configurare de utilități și tabel de indexuri.

![Pagina de proprietăți cu card-ul de prezentare](/screenshots/properties.png){width="800"}

## Adăugarea unei proprietăți

Din sidebar → **Proprietăți** → apasă butonul **+** (adaugă proprietate).

### Câmpuri disponibile

| Câmp | Obligatoriu | Descriere |
|---|---|---|
| **Nume** | ✓ | Numele proprietății (ex: „Casa Chiajna", „Apartament Centru") |
| **Tip** | ✓ | Casă, apartament, studio, garaj, birou, magazin, atelier, teren, altele |
| Adresă | ✗ | Adresa completă (stradă, număr) |
| Oraș | ✗ | Orașul |
| Județ | ✗ | Județul |
| Cod poștal | ✗ | Codul poștal |
| Suprafață | ✗ | Suprafața utilă (mp) |
| Camere | ✗ | Numărul de camere |
| Băi | ✗ | Numărul de băi |
| Descriere | ✗ | Note sau detalii suplimentare |

::alert{type="info"}
**Sfat:** Completează cel puțin orașul și județul — aceste informații apar pe card-ul de proprietate și te ajută să identifici rapid fiecare imobil.
::

## Card-ul de proprietate

Fiecare proprietate este afișată ca un **card** vizual care arată:
- Numele și locația
- Tipul și suprafața
- Starea (Activ / Închiriat / Inactiv)
- **Ultimele consumuri** — valorile recente per utilitate (Electricitate, Gaz, Apă)

Sub card, găsești 3 acțiuni rapide:
- 🔓 **Blocare/Deblocare** — previne modificările accidentale
- ✏️ **Editare** — modifică detaliile proprietății
- 🗑️ **Ștergere** — elimină proprietatea (soft delete)

## Tabelul de indexuri

Sub card-ul principal, **Tabelul de indexuri** arată consumul pe ultimele 12 luni pentru fiecare utilitate configurată:

![Tabelul de indexuri cu Electricitate, Gaz și Apă](/screenshots/property-index.png){width="800"}

Fiecare rând afișează:
- **Luna** — de la Mar 2025 până la Feb 2026 (exemplu)
- **Consum** — în kWh, m³ sau Gcal (în funcție de utilitate)
- **Factură (RON)** — costul estimat sau real

::alert{type="warning"}
Tabelul de indexuri se populează automat pe baza citirilor de contor. Dacă nu ai introdus încă citiri, vei vedea doar liniuțe (—).
::

## Statusuri proprietate

| Status | Descriere |
|---|---|
| **Activ** | Proprietate folosită în mod curent |
| **Închiriat** | Proprietate dată în chirie (pe Pro+ poți invita rezidenți) |
| **Inactiv** | Proprietate temporar neutilizată |

## Blocarea unei proprietăți

Numai **owner-ul** contului poate bloca/debloca o proprietate. O proprietate blocată:
- Poate fi vizualizată de toți membrii echipei
- **Nu** poate fi editată sau ștearsă de nimeni (inclusiv admin)
- Doar owner-ul poate debloca

## Ștergerea unei proprietăți

Ștergerea este disponibilă doar pentru rolurile **owner** și **admin**. Proprietatea nu este ștearsă definitiv — folosim **soft delete** (marcată cu data ștergerii) și poate fi recuperată de echipa de suport.

# Servicii recurente

Pe lângă utilitățile cu contor, proprietățile au și **servicii recurente** — cheltuieli fixe sau periodice care nu necesită citiri.

## Exemple de servicii

- Internet / WiFi
- Gunoi / salubritate
- Administrație bloc
- Întreținere spații verzi
- Curățenie
- Pază / securitate

## Adăugarea unui serviciu

Din pagina proprietății → **Configurare** → **Servicii**:

| Câmp | Obligatoriu | Descriere |
|---|---|---|
| Tip serviciu | ✓ | Identificatorul serviciului (ex: „wifi", „trash") |
| Etichetă | ✓ | Numele afișat (ex: „Internet Digi") |
| Furnizor | ✗ | Numele companiei |
| Nr. contract | ✗ | Numărul contractului |
| Cadență | ✓ | Frecvența: lunar, trimestrial, anual, o dată, custom |
| Suma (RON) | ✗ | Valoarea facturii |
| Note | ✗ | Informații suplimentare |

## Cadențe disponibile

| Cadență | Descriere |
|---|---|
| `one_time` | Plată unică |
| `monthly` | Lunar |
| `quarterly` | Trimestrial |
| `yearly` | Anual |
| `custom` | Interval custom (ex: la fiecare 2 luni) |

::alert{type="info"}
Pentru cadența `custom`, trebuie specificată și **unitatea** (lună, trimestru, an) și **frecvența** (la fiecare câte unități).
::

## Catalogul de servicii

CuibOS oferă un catalog predefinit de servicii care facilitează adăugarea rapidă. Catalogul conține tipuri comune (internet, gunoi, administrație) cu icoane și valori implicite.

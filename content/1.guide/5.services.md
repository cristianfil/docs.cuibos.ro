# Servicii recurente

Pe lângă utilitățile cu contor, proprietățile au și **servicii recurente** — cheltuieli fixe sau periodice care nu necesită citiri de contor.

## Exemple de servicii tipice

| Serviciu | Cadență tipică | Exemplu cost |
|---|---|---|
| Internet / WiFi | lunar | 49 RON |
| TV cablu | lunar | 35 RON |
| Gunoi / salubritate | lunar | 25 RON |
| Administrație bloc | lunar | 150 RON |
| Întreținere spații verzi | lunar | 100 RON |
| Curățenie | lunar | 200 RON |
| Pază / securitate | lunar | 80 RON |
| Asigurare locuință | anual | 500 RON |

## Adăugarea unui serviciu

Din pagina proprietății → **Configurare** → secțiunea **Servicii**:

| Câmp | Obligatoriu | Descriere |
|---|---|---|
| **Tip serviciu** | ✓ | Identificatorul (ex: „wifi", „trash", „admin") |
| **Etichetă** | ✓ | Numele afișat (ex: „Internet Digi", „Administrație bloc") |
| Furnizor | ✗ | Numele companiei |
| Nr. contract | ✗ | Numărul contractului |
| **Cadență** | ✓ | Frecvența plății |
| Suma (RON) | ✗ | Valoarea recurentă a facturii |
| Note | ✗ | Informații suplimentare |

## Cadențe disponibile

| Cadență | Descriere | Exemplu |
|---|---|---|
| **O dată** | Plată unică | Comision agenție imobiliară |
| **Lunar** | În fiecare lună | Internet, administrație |
| **Trimestrial** | La 3 luni | Impozit local |
| **Anual** | O dată pe an | Asigurare locuință |
| **Custom** | Interval personalizat | La fiecare 2 luni |

::alert{type="info"}
Pentru cadența **custom**, specifici unitatea (lună, trimestru, an) și frecvența (la câte unități). Exemplu: „la fiecare 2 luni" = unitate: lună, frecvență: 2.
::

## Catalogul de servicii

CuibOS oferă un **catalog predefinit** de servicii pentru adăugare rapidă. Catalogul conține tipurile comune (internet, gunoi, administrație, curățenie, pază) cu icoane și valori implicite.

Poți folosi catalogul ca punct de plecare și modifica detaliile după preferințe.

## Propagarea în buget

Serviciile recurente cu sumă completată se **propagă automat** în bugetul lunnar al proprietății. Astfel, bugetul „Casa 1" va conține automat liniile pentru internet, gunoi, administrație etc.

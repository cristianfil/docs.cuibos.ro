# Utilități și contoare

Fiecare proprietate poate avea mai multe utilități configurate, iar fiecare utilitate poate avea unul sau mai multe contoare.

## Tipuri de utilități

| Tip | Descriere |
|---|---|
| `electricity` | Curent electric |
| `gas` | Gaz natural |
| `water` | Apă (rece / caldă) |
| `heating` | Încălzire / termoficare |

## Configurarea utilităților

Din pagina proprietății → **Configurare** → **Utilități**:

::steps

### Adaugă utilitate

Selectează tipul (ex: electricity) și completează informațiile furnizorului:
- **Furnizor** — numele companiei (ex: Enel, E.ON)
- **Nr. contract** — numărul contractului
- **Zi start / sfârșit** — intervalul lunii pentru transmiterea citirilor

### Adaugă contoare

Fiecare utilitate poate avea mai multe contoare:
- **Etichetă** — numele contorului (ex: „Contor zi", „Contor noapte")
- **Număr contor** — seria contorului
- **Unitate** — unitatea de măsură (kWh, mc, Gcal)

### Salvează configurarea

Configurarea se salvează integral (utilități + contoare) printr-un singur request `PUT`.

::

## Citirea contoarelor

Din pagina proprietății → **Consum**:

1. Selectează contorul dorit
2. Introdu **valoarea citită** și **data citirii**
3. Marchează opțional dacă a fost **transmisă furnizorului**
4. Salvează

::alert{type="info"}
Ciclurile de consum se calculează automat pe baza citirilor consecutive. Dacă corectezi o citire veche, poți recalcula ciclurile manual.
::

## Vizualizarea consumului

Pagina de consum afișează:
- **Citiri recente** — tabelul cu ultimele citiri per contor
- **Cicluri** — consumul calculat pe perioade (luni)
- **Grafic consum** — serie temporală chart-friendly

## Tabel index

Tabelul index oferă o vedere agregată pe lunile recente a consumului per utilitate, ideal pentru comparații rapide. Accesibil din proprietate → **Tabel index**.

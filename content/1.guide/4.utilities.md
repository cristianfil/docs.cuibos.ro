# Utilități și contoare

Fiecare proprietate poate avea utilități configurate (curent, gaz, apă, încălzire), iar fiecare utilitate poate avea unul sau mai multe contoare individuale.

## Tipuri de utilități

| Tip | Iconiță | Unitate tipică |
|---|---|---|
| **Electricitate** | ⚡ | kWh |
| **Gaz** | 🔥 | m³ |
| **Apă** | 💧 | m³ |
| **Încălzire** | 🌡️ | Gcal |

## Configurarea utilităților

Din pagina proprietății → secțiunea **Configurare**:

::steps

### Adaugă utilitatea

Selectează tipul (ex: Electricitate) și completează informațiile furnizorului:
- **Furnizor** — numele companiei (ex: Enel, E.ON, Engie)
- **Nr. contract** — numărul contractului cu furnizorul
- **Zi start citire** — prima zi din lună când poți transmite citirea (ex: 1)
- **Zi sfârșit citire** — ultima zi din lună când poți transmite citirea (ex: 15)

### Adaugă contoarele

Fiecare utilitate poate avea mai multe contoare. Exemplu tipic: un contor de curent cu tarif zi + tarif noapte.

| Câmp | Descriere |
|---|---|
| **Etichetă** | Numele contorului (ex: „Contor zi", „Contor noapte") |
| **Nr. contor** | Seria contorului de pe aparat |
| **Unitate** | Unitatea de măsură (kWh, m³, Gcal) |

### Salvează configurarea

Configurarea se salvează integral (toate utilitățile + contoarele) printr-o singură acțiune.

::

::alert{type="info"}
**Sfat:** Intervalul de citire (zi start → zi sfârșit) este important — CuibOS îl folosește pentru a-ți trimite remindere de transmitere a citirilor către furnizor.
::

## Înregistrarea citirilor

Citirile contoarelor se fac din pagina proprietății, secțiunea **Consum**:

1. **Selectează contorul** din lista de utilități
2. Introdu **valoarea citită** de pe contor (ex: 15234)
3. Selectează **data citirii**
4. Bifează opțional dacă ai **transmis citirea furnizorului**
5. Adaugă **note** dacă e nevoie (ex: „citire estimată")
6. **Salvează**

::alert{type="warning"}
**Important:** Dacă introduci o citire mai mică decât precedenta, CuibOS va semnala o posibilă eroare. Verifică valoarea înainte de salvare.
::

## Cicluri de consum

CuibOS calculează automat **ciclurile de consum** pe baza citirilor consecutive:

- **Consum** = citirea curentă − citirea anterioară
- **Perioadă** = data curentă − data anterioară
- **Status ciclu**: deschis (incomplet), închis (calculat), corectat

Dacă corectezi o citire veche, poți **recalcula manual** ciclurile din meniu.

## Vizualizarea consumului

Pagina de consum oferă 3 perspective:

| Perspectivă | Descriere |
|---|---|
| **Citiri recente** | Tabel cu ultimele citiri per contor, ordonate cronologic |
| **Cicluri** | Consumul calculat per perioadă cu status și sursă |
| **Grafic** | Serie temporală vizuală a consumului în timp |

## Tabelul index

Tabelul index oferă o **vizualizare lunară agregată** a consumului per utilitate pe ultimele 12 luni. E ideale pentru:
- Compararea consumului între luni
- Identificarea tendințelor (consum crescut/scăzut)
- Verificarea rapidă a facturilor

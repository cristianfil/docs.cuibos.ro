# Recurring Services

In addition to metered utilities, properties also have **recurring services** — fixed or periodic expenses that don't require readings.

## Service examples

- Internet / WiFi
- Waste collection
- Building administration
- Landscaping
- Cleaning
- Security

## Adding a service

From the property page → **Configuration** → **Services**:

| Field | Required | Description |
|---|---|---|
| Service type | ✓ | Service identifier (e.g., "wifi", "trash") |
| Label | ✓ | Display name (e.g., "Digi Internet") |
| Provider | ✗ | Company name |
| Contract no. | ✗ | Contract number |
| Cadence | ✓ | Frequency: monthly, quarterly, yearly, one-time, custom |
| Amount (RON) | ✗ | Invoice amount |
| Notes | ✗ | Additional information |

## Available cadences

| Cadence | Description |
|---|---|
| `one_time` | One-time payment |
| `monthly` | Monthly |
| `quarterly` | Quarterly |
| `yearly` | Yearly |
| `custom` | Custom interval (e.g., every 2 months) |

::alert{type="info"}
For `custom` cadence, you must also specify the **unit** (month, quarter, year) and the **frequency** (every how many units).
::

## Service catalog

CuibOS provides a predefined service catalog to facilitate quick addition. The catalog contains common types (internet, waste, administration) with icons and default values.

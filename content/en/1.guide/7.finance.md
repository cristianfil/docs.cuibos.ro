# Finance

The finance module helps you plan and monitor family spending.

## Budgets

Create budgets to track spending by category and period.

### Creating a budget

- **Name** — budget name (e.g., "January 2025 Expenses")
- **Period** — month/year
- **Total amount** — total allocated budget

### Budget lines

Each budget contains **lines** with:
- **Category** — expense type (utilities, services, food, transport)
- **Planned amount** — how much you allocated
- **Actual amount** — how much you actually spent
- **Responsible** — the team member responsible (optional)

### Unplanned expenses

You can also add **unplanned** expenses — those that weren't in the initial budget but came up along the way.

### Budget strategy

Define the budget splitting strategy: who contributes how much, what rules apply.

## Bills

The bills section allows tracking all payment invoices:

| Field | Description |
|---|---|
| Issuer | Who issued the bill |
| Amount | Bill amount |
| Due date | Payment deadline |
| Status | unpaid / paid / overdue |
| Property | Link to property (optional) |

::alert{type="info"}
Bills approaching their due date automatically generate notifications through the reminder service.
::

## Installments

Manage monthly installments — loans, leasing, subscription payments:

- **Creating an installment** — total amount, number of installments, interval, start date
- **Paying an installment** — mark as paid
- **Activate/deactivate** — pause an installment

## Exchange rates

CuibOS provides an FX rate endpoint for currency conversion. Useful for multi-currency budgets.

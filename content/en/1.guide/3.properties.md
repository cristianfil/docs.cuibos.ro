# Properties

The properties module lets you manage all your family real estate: apartments, houses, garages, offices, land, etc.

## Adding a property

::steps

### Go to the Properties section

From the sidebar, choose **Properties** → **Add property**.

### Fill in the details

Available fields:

| Field | Required | Description |
|---|---|---|
| Name | ✓ | Property name (e.g., "Downtown Apartment") |
| Type | ✓ | house, apartment, studio, garage, office, shop, workshop, land, other |
| Address | ✗ | Full address |
| City | ✗ | City |
| County | ✗ | County / state |
| Postal code | ✗ | Postal code |
| Surface area | ✗ | Usable area (sqm) |
| Rooms | ✗ | Number of rooms |
| Bathrooms | ✗ | Number of bathrooms |
| Description | ✗ | Additional notes |

### Save

The property will appear on the dashboard card with **active** status.

::

## Property statuses

- **Active** — property currently in use
- **Rented** — property is rented out
- **Inactive** — property temporarily unused

## Locking a property

The account owner can **lock** a property to prevent accidental modifications by other team members.

::alert{type="info"}
A locked property can be viewed but cannot be edited or deleted by anyone except the owner.
::

## Deleting a property

Deletion is only available for **owner** and **admin** roles. The property is not permanently removed — it is marked as deleted (soft delete) and can be recovered by the support team.

# Reminder & Notify Services

## Reminder Service

Prefix: `/reminders` (internal, not exposed through gateway)

The reminder service orchestrates periodic due date checking and triggers notifications.

### POST /reminders/materialize

Materializes reminders for the current period — scans all resources and creates alerts.

---

### POST /reminders/dispatch

Sends materialized notifications that haven't been delivered yet.

---

### POST /reminders/recover-stale

Recovers stale reminders — those stuck in "processing" state for too long.

---

### POST /reminders/cleanup

Cleans up old / expired reminders from the database.

---

### POST /reminders/reconcile

Reconciles reminder state with the actual state of verified resources.

---

## Reminder Flow

```
┌─────────────────┐
│  CRON trigger    │  (materialize, dispatch — called periodically)
└────────┬────────┘
         │
         ▼
┌─────────────────┐     ┌──────────────────┐
│ reminder-service │────▶│ vehicles-service  │  (verify compliance/maintenance)
│                 │────▶│ finance-service   │  (verify bills)
└────────┬────────┘     └──────────────────┘
         │
         ▼
┌─────────────────┐
│  notify-service  │  (sends emails / push notifications)
└─────────────────┘
```

## Notify Service

Prefix: `/notify/v1` (gateway)

The notification service receives requests from the reminder service and delivers:
- **Emails** — via external provider (e.g., Resend)
- **In-app notifications** — via WebSocket or polling

::alert{type="info"}
The notify service is used exclusively by other microservices. There are no endpoints intended for direct user access.
::

## Alert Types

| Category | Source | Description |
|---|---|---|
| MOT expired | vehicles-service | Technical inspection approaching due date |
| Insurance expired | vehicles-service | Mandatory insurance approaching due date |
| Road tax | vehicles-service | Road tax approaching due date |
| Maintenance | vehicles-service | Maintenance item due (km or time) |
| Bill due | finance-service | Bill with approaching payment deadline |
| Meter reading | property-service | Meter reading submission interval open |

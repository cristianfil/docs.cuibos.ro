# Property Service

Prefix: `/properties`

All endpoints (except `/properties/metadata`) require a JWT.

## Properties

### POST /properties/newProperty

🔒 **Requires JWT** (admin or owner)

Creates a new property.

::field-group
  ::field{name="name" type="string" required}
  Property name (max 255 characters)
  ::
  ::field{name="propertyType" type="enum" required}
  `house` | `apartment` | `studio` | `garage` | `office` | `shop` | `workshop` | `land` | `other`
  ::
  ::field{name="address" type="string"}
  Full address
  ::
  ::field{name="city" type="string"}
  City (max 120)
  ::
  ::field{name="county" type="string"}
  County (max 120)
  ::
  ::field{name="surfaceArea" type="number"}
  Usable area (sqm)
  ::
  ::field{name="status" type="enum"}
  `active` | `rented` | `inactive` (default: `active`)
  ::
::

---

### GET /properties/properties

🔒 Returns all properties for the tenant.

---

### GET /properties/metadata

Returns available enums (types and statuses). **Does not require JWT.**

---

### GET /properties/cards/available

🔒 Returns property cards (dashboard view).

| Query param | Type | Default | Description |
|---|---|---|---|
| `take` | number | 50 | Number of results (1-200) |

---

### GET /properties/:propertyId

🔒 Returns complete property details with all relations.

---

### PUT /properties/:propertyId

🔒 **(admin/owner)** Updates property details.

---

### PATCH /properties/:id/lock

🔒 **(owner only)** Locks/unlocks a property.

---

### DELETE /properties/:id

🔒 **(admin/owner)** Deletes a property (soft delete).

---

## Configuration (utilities + services)

### GET /properties/:propertyId/config

🔒 Returns the complete property configuration.

---

### PUT /properties/:propertyId/config

🔒 **(admin/owner)** Fully updates the configuration (utilities + services in one request).

---

### GET /properties/services/catalog

🔒 Returns the available service catalog.

---

## Meters & Readings

### POST /properties/utilities/:propertyUtilityId/meters

🔒 Adds a new meter to a utility.

---

### PATCH /properties/meters/:meterId/readings

🔒 **(admin/owner)** Adds or updates a meter reading (upsert on meterId + readingDate).

::field-group
  ::field{name="value" type="string" required}
  Reading value (decimal)
  ::
  ::field{name="readingDate" type="string" required}
  Reading date (ISO date)
  ::
  ::field{name="readingType" type="string" required}
  Reading type
  ::
  ::field{name="submittedToProvider" type="boolean"}
  Whether it was submitted to the provider
  ::
::

---

### GET /properties/meters/:meterId/readings

🔒 Returns meter readings (paginated, most recent first).

---

### GET /properties/meters/:meterId/cycles

🔒 Returns calculated consumption cycles.

---

### PATCH /properties/meters/:meterId/cycles/recompute

🔒 **(admin/owner)** Recalculates the meter's consumption cycles.

---

## Consumption

### GET /properties/:propertyId/consumption

🔒 Returns full property consumption (readings + cycles + chart series per meter).

---

### GET /properties/:propertyId/index-table

🔒 Index table — monthly aggregated consumption per utility.

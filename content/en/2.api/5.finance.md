# Finance Service

Prefix: `/finance/v1` (gateway) → `/finance` (internal)

All endpoints require JWT.

## Budgets

### POST /finance/v1/budgets

🔒 Creates a new budget.

---

### GET /finance/v1/budgets

🔒 Returns budgets for the tenant.

---

### GET /finance/v1/budgets/:id

🔒 Returns budget details.

---

### PATCH /finance/v1/budgets/:id

🔒 Updates a budget.

---

### DELETE /finance/v1/budgets/:id

🔒 Deletes a budget.

---

## Budget Lines

### POST /finance/v1/budgets/:id/lines

🔒 Adds a budget line (planned expense).

---

### POST /finance/v1/budgets/:id/lines/unplanned

🔒 Adds an unplanned expense to the budget.

---

### PATCH /finance/v1/budgets/:id/lines/:lineId

🔒 Updates a budget line.

---

### DELETE /finance/v1/budgets/:id/lines/:lineId

🔒 Deletes a budget line.

---

## Budget Strategy

### PUT /finance/v1/budgets/:id/strategy

🔒 Defines/updates the budget strategy (contribution rules, limits).

---

## User Directory

### GET /finance/v1/users/directory

🔒 Returns the list of team members (for expense assignment).

---

## Bills

### GET /finance/v1/bills

🔒 Returns bills for the tenant.

---

### POST /finance/v1/bills

🔒 Creates a new bill.

---

### DELETE /finance/v1/bills/:billId

🔒 Deletes a bill.

---

### POST /finance/v1/internal/reminders/bills/verify

Internal endpoint — checks bills approaching their due date for notifications.

---

## Installments

### GET /finance/v1/installments

🔒 Returns installments for the tenant.

---

### POST /finance/v1/installments

🔒 Creates a new installment.

---

### PATCH /finance/v1/installments/:id

🔒 Updates an installment.

---

### POST /finance/v1/installments/:id/pay

🔒 Marks an installment as paid.

---

### POST /finance/v1/installments/:id/toggle-active

🔒 Activates/deactivates an installment.

---

## Exchange Rates

### GET /finance/v1/fx/rate

🔒 Returns current exchange rate.

| Query param | Type | Description |
|---|---|---|
| `from` | string | Source currency (e.g., `EUR`) |
| `to` | string | Target currency (e.g., `RON`) |

# Property Service

Prefix: `/properties`

Toate endpoint-urile (cu excepția `/properties/metadata`) necesită JWT.

## Proprietăți

### POST /properties/newProperty

🔒 **Necesită JWT** (admin sau owner)

Creează o nouă proprietate.

::field-group
  ::field{name="name" type="string" required}
  Numele proprietății (max 255 caractere)
  ::
  ::field{name="propertyType" type="enum" required}
  `house` | `apartment` | `studio` | `garage` | `office` | `shop` | `workshop` | `land` | `other`
  ::
  ::field{name="address" type="string"}
  Adresa completă
  ::
  ::field{name="city" type="string"}
  Orașul (max 120)
  ::
  ::field{name="county" type="string"}
  Județul (max 120)
  ::
  ::field{name="postalCode" type="string"}
  Codul poștal (max 30)
  ::
  ::field{name="surfaceArea" type="number"}
  Suprafața utilă (mp)
  ::
  ::field{name="rooms" type="number"}
  Numărul de camere
  ::
  ::field{name="bathrooms" type="number"}
  Numărul de băi
  ::
  ::field{name="description" type="string"}
  Descriere / note
  ::
  ::field{name="status" type="enum"}
  `active` | `rented` | `inactive` (default: `active`)
  ::
::

---

### GET /properties/properties

🔒 Returnează toate proprietățile tenant-ului.

---

### GET /properties/metadata

Returnează enum-urile disponibile (tipuri și statusuri). **Nu necesită JWT.**

```json
{
  "propertyTypes": ["house", "apartment", "studio", ...],
  "propertyStatuses": ["active", "inactive", "rented"]
}
```

---

### GET /properties/cards/available

🔒 Returnează card-urile proprietăților (vizualizare dashboard).

| Query param | Tip | Default | Descriere |
|---|---|---|---|
| `take` | number | 50 | Număr rezultate (1-200) |

---

### GET /properties/:propertyId

🔒 Returnează detaliile complete ale unei proprietăți cu toate relațiile.

---

### PUT /properties/:propertyId

🔒 **(admin/owner)** Actualizează detaliile proprietății.

---

### PATCH /properties/:id/lock

🔒 **(owner only)** Blochează/deblochează proprietatea.

::field-group
  ::field{name="locked" type="boolean" required}
  `true` pentru blocare, `false` pentru deblocare
  ::
::

---

### DELETE /properties/:id

🔒 **(admin/owner)** Șterge proprietatea (soft delete).

---

## Configurare (utilități + servicii)

### GET /properties/:propertyId/config

🔒 Returnează configurarea completă a proprietății (utilități + servicii).

---

### PUT /properties/:propertyId/config

🔒 **(admin/owner)** Actualizează integral configurarea.

Corpul request-ului conține:

```json
{
  "utilities": [
    {
      "utilityType": "electricity",
      "providerName": "Enel",
      "contractNumber": "C123",
      "readingSubmissionDayStart": 1,
      "readingSubmissionDayEnd": 15,
      "meters": [
        { "meterLabel": "Contor zi", "meterNumber": "SN123", "unit": "kWh" }
      ]
    }
  ],
  "services": [
    {
      "serviceType": "wifi",
      "instanceLabel": "Internet Digi",
      "providerName": "Digi",
      "cadence": "monthly",
      "invoiceAmountRon": "49.99"
    }
  ]
}
```

---

### GET /properties/services/catalog

🔒 Returnează catalogul de servicii disponibile.

---

## Contoare și citiri

### POST /properties/utilities/:propertyUtilityId/meters

🔒 Adaugă un contor nou la o utilitate.

---

### PATCH /properties/meters/:meterId/readings

🔒 **(admin/owner)** Adaugă sau actualizează o citire de contor (upsert pe meterId + readingDate).

::field-group
  ::field{name="value" type="string" required}
  Valoarea citită (decimal)
  ::
  ::field{name="readingDate" type="string" required}
  Data citirii (ISO date)
  ::
  ::field{name="readingType" type="string" required}
  Tipul citirii
  ::
  ::field{name="submittedToProvider" type="boolean"}
  Dacă a fost transmisă furnizorului
  ::
  ::field{name="notes" type="string"}
  Note suplimentare
  ::
::

---

### GET /properties/meters/:meterId/readings

🔒 Returnează citirile unui contor (paginat, cele mai recente primele).

| Query param | Tip | Default | Descriere |
|---|---|---|---|
| `take` | number | 50 | Număr rezultate (1-500) |

---

### GET /properties/meters/:meterId/cycles

🔒 Returnează ciclurile de consum calculate.

| Query param | Tip | Default | Descriere |
|---|---|---|---|
| `take` | number | 24 | Număr cicluri (1-500) |

---

### PATCH /properties/meters/:meterId/cycles/recompute

🔒 **(admin/owner)** Recalculează ciclurile de consum ale contorului.

---

## Consum

### GET /properties/:propertyId/consumption

🔒 Returnează consumul complet al proprietății (citiri + cicluri + serie grafic per contor).

| Query param | Tip | Default | Descriere |
|---|---|---|---|
| `takeReadings` | number | 12 | Citiri per contor |
| `takeCycles` | number | 12 | Cicluri per contor |

---

### GET /properties/:propertyId/index-table

🔒 Tabelul index — consum agregat lunar per utilitate.

| Query param | Tip | Default | Descriere |
|---|---|---|---|
| `months` | number | 12 | Număr luni (1-24) |

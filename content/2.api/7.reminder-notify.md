# Reminder & Notify Services

## Reminder Service

Prefix: `/reminders` (intern, nu expus prin gateway)

Serviciul de reminder-e orchestrează verificarea periodică a scadențelor și declanșează notificări.

### POST /reminders/materialize

Materializează reminder-ele pentru perioada curentă — scanează toate resursele și creează alerte.

---

### POST /reminders/dispatch

Trimite notificările materializate care nu au fost încă livrate.

---

### POST /reminders/recover-stale

Recuperează reminder-ele blocate (stale) — cele care au rămas în starea „processing" prea mult timp.

---

### POST /reminders/cleanup

Curăță reminder-ele vechi / expirate din baza de date.

---

### POST /reminders/reconcile

Reconciliază starea reminder-elor cu starea reală a resurselor verificate.

---

## Fluxul de reminder-e

```
┌─────────────────┐
│  CRON trigger    │  (materialize, dispatch — apelate periodic)
└────────┬────────┘
         │
         ▼
┌─────────────────┐     ┌──────────────────┐
│ reminder-service │────▶│ vehicles-service  │  (verify compliance/maintenance)
│                 │────▶│ finance-service   │  (verify bills)
└────────┬────────┘     └──────────────────┘
         │
         ▼
┌─────────────────┐
│  notify-service  │  (trimite email-uri / push notifications)
└─────────────────┘
```

## Notify Service

Prefix: `/notify/v1` (gateway)

Serviciul de notificări primește cereri de la reminder-service și livrează:
- **Email-uri** — prin provider extern (ex: Resend)
- **Notificări in-app** — prin WebSocket sau polling

::alert{type="info"}
Notify-service este folosit exclusiv de alte microservicii. Nu există endpoint-uri destinate direct utilizatorilor.
::

## Tipuri de alerte

| Categorie | Sursă | Descriere |
|---|---|---|
| ITP expirat | vehicles-service | Inspecția tehnică periodică aproape de scadență |
| RCA expirat | vehicles-service | Asigurarea obligatorie aproape de scadență |
| Rovinieta | vehicles-service | Taxa de drum aproape de scadență |
| Mentenanță | vehicles-service | Element de mentenanță scadent (km sau timp) |
| Factură scadentă | finance-service | Factură cu termen de plată apropiat |
| Citire contor | property-service | Interval de transmitere citiri deschis |
